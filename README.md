### ⚙️ Installation

- Requires: `Python 3.9+`
- Install the requirements: `pip install -r requirements.txt`
- Start: `python3 main.py`

---

### 🔥 Features
  - Custom RPC
  - Super Fast
  - Simple & easy to use
  - Threading
  - Music
  - Auto proxy scraping
  - Asks if we want to make infinite links or a specific amount
  - More but I forgot since I made it in February 🤓
